springboot热部署的实现方式

使用spring-boot-devtools

  1. 添加pom依赖
  2. 配置idea
    shift+ctrl+alt+”/“ （IDEA中的快捷键）
    选择”Registry”
    勾选 compiler.automake.allow.when.app.running

使用spring loaded

  1. 添加pom插件
  2. 在idea中，修改settings->compiler，勾选Build project automatically.
  3. 在idea中，使用右侧maven的plugins，找到sping-boot:run命令双击执行即可启动服务
