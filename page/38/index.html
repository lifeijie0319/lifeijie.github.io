Lombok

发表于 2022-09-12 | 分类于 language ， java ， lib

简介

Lombok是一个通过注解简化getter，setter等重复代码编写的工具

常用注解

@Sl4j注解

@Sl4j注解注释在类上，代替自己手动写logger，注解后即可在代码中使用log作为日志打印对象

参考资料

https://blog.csdn.net/qq_43842093/article/details/121257969

常见问题

idea中Lombok注解不起作用

  1. 添加Lombok注解
  2. 在idea的plugins里下载安装lommbox
    如果出现not found或持续加载不出现内容那么我们需要在Setting–》Appearance & Behavior –》Syetem Setting –》Updates，将Use secure connection 的勾选去掉，然后删除idea的进程（任务管理器），重启再搜索即可。
    部分idea版本里没有Use secure connection 、automatic proxy 等选项，那在plugins点开设置项，点击HTTP Proxy settings，勾中Auto-detect proxy settings和Automatic proxy，并在后面填入plugins.jetbrains.com，重启安装

    参考资料

    https://blog.csdn.net/m0_52937388/article/details/120090292
