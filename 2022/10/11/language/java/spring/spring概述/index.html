什么是spring

Spring是一个轻量级的控制反转(IoC)和面向切面(AOP)的容器框架。

轻量

  1. 大小，完整的spring框架可以在一个1M多的文件里发布。
  2. 开销小。
  3. 非侵入式，应用的类不依赖于spring框架的类。

    控制反转

    对象之间的依赖关系不再由应用代码显式创建，而是由spring根据配置和注解自动注入。即A依赖B的关系由原来的A主动创建B变成了由spring框架来注入B，依赖的建立由主动变成被动。

    面向切面

    spring提供了面向切面的丰富支持，使得应用本身的逻辑和系统级服务（日志、事务处理等）分离开来，方便复用，降低耦合。

    容器

    spring管理应用对象的整个生命周期，所以是一个容器。

    框架

    spring内置了很多功能，并且也容易与其他功能集成，使得开发者能在spring的基础上快速开发应用。

spring模块

  • Core Container: Beans Core Context SpEL
  • AOP: Aspects Instruments Messageing
  • Web: Web WebSocket Servlet Portlet
  • Data Access/Integration: JDBC ORM OXM JMS Transactions
  • Test
