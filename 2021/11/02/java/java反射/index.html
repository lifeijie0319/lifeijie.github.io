java反射机制的意义

参考：https://www.zhihu.com/question/20157023/answer/1021842505
按照上述参考回答，反射的意义有二：

  • 提高程序的灵活性
  • 屏蔽掉实现的细节，让使用者更加方便好用

第一点文章以数据库驱动的动态加载为例说明，但是这里一直有疑惑。为什么不是不同的数据库驱动厂商实现统一接口，而使用者调用接口即可？
第二点以springMVC框架的参数自动装载为例说明，到是没有什么疑义。
