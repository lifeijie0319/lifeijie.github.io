pygraphviz安装

发表于 2021-03-25

介绍

Graphviz 是一个几何图形可视化软件，能够将数据的结构信息表示为抽象图形和网络图的方式。PyGraphviz 主要对数据量大、可任意定制的图形上具有强大优势。

pygraphviz安装问题

安装方法：

python setup.py install

现象：

安装pygraphviz失败

解决：

需要先安装graphviz和pygraphviz-devel。因为yum没有配置，所以需要先在官网下载yum源配置文件。之后可以用yum安装。然而yum源安装获取的版本（2.40）与网站提供的版本（2.38）不一致，所以需要先下载rpm包，然后sudo yum localinstall graphviz-devel-2.38.0-1.el7.x86_64.rpm graphviz-2.38.0-1.el7.x86_64.rpm

参考：

http://www.cnblogs.com/rocketfan/archive/2009/09/09/1563628.html

注：在ubuntu下面安装 graphviz

sudo apt-get install graphviz, graphviz-dev
注意graphiviz-dev是pygraphviz所依赖的必须安装。
pip install pygraphviz
