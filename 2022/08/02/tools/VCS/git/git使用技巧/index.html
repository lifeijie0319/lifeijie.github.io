git使用技巧

发表于 2022-08-02

查看从某个提交开始修改了哪些文件

查看最后一次提交记录的修改文件信息

git show –raw

查看指定commit id对应修改文件列表

git show –raw commit_id

查看所有提交记录的修改文件信息

git log –stat
git log –name-only

查看所有修改相关的commit ID和comment信息

git log –pretty=oneline

git识别文件大小写

查看配置

git在windows下默认忽略大小写，尽量不要写同名但是大小写不一样的两个文件，会造成这两个文件查看当前目录git status永远只有一个文件处于被修改状态且无法通过checkout恢复。只能通过改名修复。
