跨域

发表于 2022-02-24

什么是跨域

浏览器不允许请求当前网站域名之外的资源，跨越这个限制就是跨域

如何实现跨域

script

script标签可以得到从其他来源数据，这也是JSONP依赖的根据。

JSONP

利用script标签可以跨域，服务端返回js代码，js代码在浏览器执行。浏览器本身的js会定义接收函数比如func，然后服务端返回的一般是json数据和浏览器这个函数的拼接内容比如’func(“json数据”)’，这就是JSONP命名的缘由：JSON+padding。

CORS

Cross-Origin Resource Sharing（CORS）跨域资源共享是一份浏览器技术的规范，提供了 Web 服务从不同域传来沙盒脚本的方法，以避开浏览器的同源策略，确保安全的跨域数据传输。

代理

同源策略是浏览器端的限制，服务端没有这个问题，可以依靠服务端越过这个限制。
请求->DomainA服务器->DomainB服务器
