复制目录与复制文件

CMD

CMD执行指令时要注意，有两种格式：

注意如果使用可执行文件或命令，别忘了加双引号

  • 如Dockerfile中出现多个CMD，则只有最后一个被执行
  • 如容器启动时附加指令，则CMD被忽略。比如docker run -it nginx /bin/bash会导致CMD不执行

USER

用于指定执行后续命令的用户和用户组，USER指的是docker容器里的用户和用户组，用户和用户组必须事先存在

VOLUME

在宿主机上创建匿名卷，一般是/var/lib/docker/volumes/目录下随机创建。

WORKDIR

WORKDIR用于指定工作目录，该目录必须事先存在。
docker run -w 可以覆盖WORKDIR
