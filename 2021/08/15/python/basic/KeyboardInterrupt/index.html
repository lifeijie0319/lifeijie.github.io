理解KeyboardInterrupt

发表于 2021-08-15 | 分类于 python ， basic

KeyboardInterrupt是什么

KeyboardInterrupt是python内置的一个异常。当python程序收到SIGINT信号后，程序抛出该异常。

KeyboardInterrupt和Ctrl+C的关系

键盘操作Ctrl+C会被shell解析，发送SIGINT信号到当前shell上的前台进程。其它常见的有：

  • Ctrl+\ - SIGQUIT
  • Ctrl+Z - SIGTSTP

为什么使用kill向后台进程发送信号SIGINT不会触发KeyboardInterrupt

原理

因为SIGINT对应的默认处理方法没有注册。这是由python解释器决定的，当程序开始时，如果信号掩码对SIGINT的处理方式是SIG_IGN（代表忽略），默认的处理方法就不会进行注册。对于SIG_IGN类型的信号掩码，当前进程是继承父进程的，而父进程中注册了处理方法的信号，到了子进程后会回复默认处理方法。
