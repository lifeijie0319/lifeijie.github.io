unicode概念

Unicode（统一码、万国码、单一码）是计算机科学领域里的一项业界标准,包括字符集、编码方案等。Unicode 是为了解决传统的字符编码方案的局限而产生的，它为每种语言中的每个字符设定了统一并且唯一的二进制编码，以满足跨语言、跨平台进行文本转换、处理的要求。1990年开始研发，1994年正式公布。

简单来说，unicode是一种抽象字符到二进制编码的映射规则。

unicode基础

  • unicode也称为UCS(Universal Multiple-Octet Coded Character Set)
  • unicode每个字符对应的编码数字称为code point
  • unicode的code point的范围是 0 - 10FFFF.
  • 一个unicode的code point的写法通常是”U+”接上16进制表示的代码点数字
  • unicode的前256个字符和ISO-8859-1（即Latin-1）一样，也就是ASCII

  • scalar values在电脑中的表示就是我们实际使用的编码方式，统一称为UTF(Unicode Transformation Format)，常用的有UTF-8/UTF-16/UTF-32等。这些编码方式都能表示所有unicode的scalar values，但是各自表示一个code point的code unit不同。
