使用kerl安装erlang环境

发表于 2021-05-30 | 分类于 erlang

参考： kancloud.cn

kerl说明

kerl是一个OTP版本管理工具，可以在不同的OTP版本间随意切换。

安装kerl

curl -O https://raw.githubusercontent.com/kerl/kerl/master/kerl&chmod a+x kerl

查看所有可用的版本

./kerl list releases

下载编译

./kerl build $版本

安装OTP

./kerl install $版本 $安装路径

激活

. $安装路径/activate

取消激活

kerl_deactivate

删除

./kerl delete installation/build $版本
