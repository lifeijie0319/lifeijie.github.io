虚拟机中的Docker容器不能访问外网域名的问题

发表于 2021-12-04 | 分类于 tools ， docker

场景

按照官网getting-started教程，创建Dockerfile，然后尝试创建image失败

原因分析

docker创建的容器无法解析域名

解决尝试

方案一

方案

修改DNS，修改/etc/resolv.conf
![](/blog/images/tools/docker/docker1.png)

结果

无效

结论

不是DNS的问题

方案二

方案

更改docker网络连接模式，即在docker run 指令后加参数 --network=host，host模式的含义是使docker的网络连接模式与宿主机的网络连接模式一致。
![](/blog/images/tools/docker/docker2.png)

结果

成功

结论

但是每次访问外网需要加--network=host参数，不方便。
