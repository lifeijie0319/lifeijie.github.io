常见配置项解析

session-factory中的property属性

  • show_sql 是否在控制台打印sql
  • format_sql 在控制台打印sql是否进行格式化
  • hibernate.hbm2ddl.auto 建表策略，常见的选项有：
    • create 每次删表重建
    • update 在现有表的基础上进行更新
  • hibernate.default_schema 默认的数据库schema名称
