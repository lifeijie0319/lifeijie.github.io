maven常识

发表于 2022-10-11 | 分类于 language ， java ， maven

maven是什么

maven是java包管理工具，通过pom.xml配置文件可以指明项目依赖的jar包，并且可以通过maven提供的工具对这些依赖包进行管理。

maven库如何管理

maven库默认放在用户目录下的.m2/repository/下面，这个路径可以通过配置文件进行配置。

maven配置文件

路径

maven配置文件默认路径为maven安装目录下的conf/settings.xml，可以创建用户目录下的.m2/settings.xml，优先级更高。

配置项

pluginGroups

搜索插件时, 如果 groupId 没有显式提供时, 则以此处配置的 groupId 为默认值, 可以简单理解为默认导入这些 groupId 下的所有 artifact（需要时才下载）

pom.xml

dependencyManagement

dependencyManagement标签用于统一管理子项目的依赖，一般在父项目中声明dependencyManagement，指定某个依赖的版本，然后所有子项目通过继承依赖于该库的同一版本。注意，父项目中dependencyManagement只是声明了依赖版本关系，并不会下载依赖，子项目中明确在中添加之后，才会下载该依赖，子项目中不用指定依赖的版本，使用的是父项目dependencyManagement指定的依赖版本，如果子项目指定了版本，会覆盖父项目的。
