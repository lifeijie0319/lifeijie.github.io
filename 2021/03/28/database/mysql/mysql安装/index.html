mysql安装

发表于 2021-03-28 | 分类于 database ， mysql

一、安装

  • centos下使用yum安装
    sudo yum install -y mysql-server mysql mysql-devel

二、设置开机自启动

设置：sudo chkconfig mysqld on
查看：chkconfig –list | grep mysqld

三、启动

sudo service mysqld start

四、设置数据库root用户密码

（一）、新密码：/usr/bin/mysqladmin -u root password ‘new-password’
（二）、重设密码：
1、mysqladmin -u root -p password ‘new-password’
2、# mysql

五、mysql的目录设置

1./etc/my.cnf 这是mysql的主配置文件
2./var/lib/mysql mysql数据库的数据库文件存放位置
3./var/log mysql数据库的日志输出存放位置
