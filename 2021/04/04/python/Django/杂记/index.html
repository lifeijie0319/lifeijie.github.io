三、中文编码问题

python2.7不能直接支持Unicode，解决办法：
from django.utils.encoding import python_2_unicode_compatible
在模型类定义之前加上装饰器：
@python_2_unicode_compatible

四、将从数据库获取的queryset转化成json返回。

五、解决由于csrf保护启用导致的图片上传bug。

1、ajax头

xhr.setRequestHeader(“X-CSRFToken”, get_cookie(‘csrftoken’));

2、post中传cookie

task.addData(‘csrfmiddlewaretoken’, get_cookie(‘csrftoken’))

六、request.POST取不到数据的原因

只有ajax请求添加参数’Content-Type’：’application/x-www-form-urlencoded’才会填充request.POST。
