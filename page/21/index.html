postgresql安装与常见问题

发表于 2021-05-30 | 分类于 database ， postgresql

参考文档

https://www.postgresql.org/download/linux/redhat/
http://www.ruanyifeng.com/blog/2013/12/getting_started_with_postgresql.html

安装方式

sudo yum install postgresql-server postgresql

报错与解决

1

现象：

解决：sudo yum install python-devel

2

现象：

解决：sudo yum install postgresql-devel

3

现象：启动postgresql报错

原因：没有初始化数据库
解决：/usr/pgsql-9.2/bin/postgresql92-setup initdb

4

现象：Django连接postgresql出错：
django.db.utils.OperationalError: FATAL: Ident authentication failed for user “lifeijie”
原因：连接权限受限制
解决：
（1）在/var/lib/…./pg_hba.conf文件中添加一行：host all all 0.0.0.0/0 trust，其它所有的认证方式改成trust。
（2）重启postgresql
