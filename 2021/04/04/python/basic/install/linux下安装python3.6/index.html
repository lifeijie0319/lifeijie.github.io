linux下安装python3.6

发表于 2021-04-04 | 分类于 python ， basic ， install

一、安装

二、建立虚拟环境

python3 -m venv

三、问题与解决

1、报错：Can’t connect to HTTPS URL because the SSL module is not available

描述：python2能正常import ssl，python3不行
解决：https://www.cnblogs.com/mlgjb/p/7976561.html
总结：无法导入ssl的时候，首先检查一下，是否安装了openssl，再检查一下是否安装openssl的开发包，切换到Modules目录，修改Setup文件；取消ssl部分的注释，重新编译安装python。

2、命令行方向键乱码

解决：
方法一：
安装readline-devel
yum -y install readline-devl
重新安装python
方法二：切换到Modules目录，修改Setup文件；取消#readline readline.c -lreadline -ltermcap行的注释符
重新安装python
