tuxedo常见问题处理

发表于 2022-07-25 | 分类于 tools ， tuxedo

问题一

CMDTUX_CAT:821: INFO: Duplicate server.
原因：tux进程未正常关闭。
备注：使用kill -9杀死tux进程后重启报该错。

问题二

tmshutdown: internal error: CMDTUX_CAT:766: ERROR: must run on master node
原因：进程已经不在了，但是IPC资源还没释放；
备注：当时kill -9杀的后台进程。
解决：执行ipcs可以看到没有释放的IPC资源；执行tmipcrm -y释放tuxedo IPC资源；再用ipcs检查，如果还有，执行ipcrm删除；

问题三

tmshutdown: internal error: CMDTUX_CAT:764: ERROR: can't attach to BB
原因：服务没有启动。

加载配置文件异常

报错：CMDTUX_CAT:190: ERROR: SRVID :sevid is not unique among server group :group
原因：设置的进程由MIN/MAX决定进程数量，不同进程间需要隔开足够的距离
