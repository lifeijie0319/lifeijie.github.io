函数式编程

发表于 2022-10-10 | 分类于 language ， java ， basic

什么是函数式编程

Java没有函数，但是方法的作用和函数类似。函数式编程就是以函数为核心，可以把一个函数当作另一个函数的入参和返回值的一种编程方式。

lambda表达式

函数式编程起源于早期数学家研究的一套函数变换逻辑，称为λ-Calculus，所以常把函数式编程称为lambda计算。

Stream

什么是Stream

Stream是java8引入的一种特殊的API，它惰性地保存了一个java对象地序列。它最大地特点在于惰性计算，即对Stream地很多操作并不是在实际地数据上操作，而只是定义了一个操作步骤，最终才在某些特定方法地作用下输出结果。由于Stream惰性计算地特点，可以用来表示无穷序列。另外，Stream支持链式操作。

基本运作流程

  1. 创建Stream
  2. 将stream转换成其它类型的stream，中间存储了转换的规则
  3. 通过惰性计算或者输出集合的方法获取最终想要的结果

    常用stream转换规则

  • map 对stream的每个对象进行同样的自定义规则处理，返回的对象构成新的stream
  • filter 对stream的每个对象进行布尔判断，只保留结果为true的，构成新的stream
  • reduce 对之前计算的结果和当前值进行处理，结果作为下一次迭代的参数进入计算，最终得到的不是新的stream，是聚合计算的结果
