新用户创建问题

发表于 2021-12-04 | 分类于 tools ， shell

问题描述

新建一用户，切换到该用户下，提示符仅仅是一个$，很奇怪。输入命令，用TAB键也无法补全命令。并且没有命令历史功能。

原因查找

  1. 查看用户SHELL变量，发现用户使用的是/bin/sh
  2. 查看/bin/sh，发现这个软链接实际指向dash

    问题解决

  3. 当前问题解决
    修改/etc/passwd中相应用户的默认shell
  4. 一劳永逸的方法 修改$SHELL = /bin/bash
